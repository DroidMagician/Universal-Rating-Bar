## 1.0.0

* Initial release with icon, asset, and custom widget rendering modes.
* Half and fractional rating support with dynamic clip rendering.
* Built-in `Icons.star_half` for default star icon mode.
* Fractional icon clipping for non-half precisions (e.g. `3.7`).
* Tap and drag gestures with configurable precision.
* Read-only mode via `isInteractive` and `ignoreGestures`.
* Optional rating change animation and scale effect.
* Vertical layout support.
* `itemBuilder` API with `RatingState` enum.
* `FlexRatingBar` typedef alias.
* pub.dev topics, screenshots, and full example app.
