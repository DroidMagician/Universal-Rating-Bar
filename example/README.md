# universal_rating_bar example

Run the full interactive demo:

```bash
flutter pub get
flutter run
```

## What's included

- Default stars with auto `Icons.star_half`
- Fractional precision (`0.1`)
- Heart icon mode
- SVG asset mode (bundled star icons)
- Emoji widget mode
- `itemBuilder` mode
- Read-only display
- Vertical layout
- Drag + animation + scale effect

## Capturing screenshots

Run the app on a device or simulator, then capture frames for README updates:

```bash
# macOS simulator example
flutter run -d macos
# Use your OS screenshot tool, or:
xcrun simctl io booted screenshot ../screenshots/capture.png
```
